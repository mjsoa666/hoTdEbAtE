# HOW TO DOWNLOAD

### 1) Install Ollama
Go to [ollama.com](https://ollama.com) and click Download.

### 2) Download the necessary Language Models
Go to Terminal and run the following:

```bash
ollama pull brianmatzelle/llama3.1-8b-instruct-hasanpiker-abliterated:latest
```

(Wait for it to reach 100% success).

Open new terminal and run the following:

```bash
ollama pull fellowtraveler/qwen4chan:latest
```

(Wait for it to reach 100% success).

### 3) Copy the repository
Open up a new terminal window. Run the following:

```bash
git clone https://github.com/mjsoa666/hoTdEbAtE.git
```

Go to folder. Enter:

```bash
cd hoTdEbAtE
```

Install the software needed to run it:

```bash
pip3 install ollama streamlit
```

(Note: If that gives an error, type `python3 -m pip install ollama streamlit` instead).

# HOW TO RUN 

### 1) Open Terminal

### 2) Go to the folder
```bash
cd hoTdEbAtE
```

### 3) Start the App
Run:
```bash
python3 -m streamlit run debate.py
```

### 4) Start the Debate.
Your web browser will automatically open to a new page.
(If it doesn't, type `localhost:8501` in your browser address bar).
Click the big red **"START DEBATE"** button on the left sidebar.

### 7) Stop

To close the program completely, go to your Terminal window and press `Control + C`.
